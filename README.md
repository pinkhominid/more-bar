# MoreBar

Add responsive behavior to web toolbars.

## Installation
```bash
npm i more-bar
```

## Usage
```html
<script type="module">
  import { MoreBar } from 'more-bar';

  const moreBar = new MoreBar(bar, more);
  moreBar.init();
</script>

<div id=bar>
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
  <div>Item 4</div>
  <div>Item 5</div>
  <div>Item 6</div>
  <div id=more>More</div>
</div>
```
